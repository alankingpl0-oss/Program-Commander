# Program-Commander
Stary program, który rozwijałem od dnia liczby pi 2025 do około października 2025. Nadaje się na dyskietki. Ostatnia aktualizacja była jednak w marcu 2026, ale tutaj poprawiłem malutki bład.

**Poszególne wersje są w katalogach oznaczonymi numerami wersji (np. v3, v2, v1).**

# Ważne
Nie wszystkie wersje się zachowały. Niektóre zostały nadpisane, usunięte lub zagubione. Sytuacja podobna, jak z Unicsem z 1969, gdzie zachowały się tylko strzępy kodu.

Ten program to abandonware.
